3D Asteroids Remake

Project Type:Personal/Hobbyist
Languages/Libraries:C++, OpenGL, SFML, GLEW
Software:Microsoft Visual Studio, DeleD
Status:Ongoing, Early Alpha Complete
Playable Demo:Download for Windows


The goal of this project is to create a 3D version of Asteroids, the classic game by Atari. I am developing the game completely from scratch using C++, OpenGL, SFML, and GLEW. I am using DeleD by Delgine to create the Asteroid models. I have implemented a tool in C# which reads in (OBJ) models and exports the models to the file format used by the game.

The above video shows a demo run of the game in its current state. A playable demo for Windows is available here. Mac and Linux versions will eventually be available.


I have implemented first person camera movement, collision detection and visibility culling into the game. The most complex feature thus far is the mesh splitting that occurs when an asteroid is hit with the basic weapon. I plan on adding more features including more weapons, sound effects, per-pixel lighting and particle effects. I plan on posting source code after the next iteration.
